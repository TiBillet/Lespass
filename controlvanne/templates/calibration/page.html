{# Mode opératoire #}
{% translate "Operating procedure" %}
  1. Désactivez la tireuse dans l'admin (si pas déjà fait)
  2. Cliquez ↺ Nouvelle série pour repartir à zéro
  3. Connectez un tuyau propre sur la tireuse
  4. Présentez une carte maintenance → la vanne s'ouvre
  5. Versez dans un verre gradué jusqu'à ~50 cl
  6. Retirez la carte → la vanne se ferme
  7. Répétez les étapes 4–6 au moins 3 fois — les versements apparaissent automatiquement ci-dessous
  8. Saisissez les volumes lus sur le verre pour chaque versement
  9. Cliquez ✓ Calculer et appliquer le facteur
  10. Présentez à nouveau votre carte maintenance sur le lecteur puis retirez-la — le Pi récupère immédiatement le nouveau facteur de calibration
