# Resource List

**URL:** `GET /booking/`
**Access:** Public

## What the user sees

A list of all bookable resources by name.

Resources with no group are listed first, in alphabetical order.

Then grouped resources appear, one section per group. Groups are
ordered alphabetically by name. Within each group, resources are
also ordered alphabetically.

When a resource has a description, a short text appears below its name.

## What the user can do

Click a resource name → goes to the resource detail page.

## Edge cases

**No resources configured:** a message "Aucune ressource disponible."
