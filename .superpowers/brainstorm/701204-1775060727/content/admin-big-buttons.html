Gros boutons admin par tenant

Chaque tenant administrable = un bouton rouge avec nom + adresse. Remplace le bouton unique "Panneau d'administration".

Contexte

L'utilisateur est admin de 2 tenants (Festival Raffinerie + Café associatif) mais pas du tenant courant (TiBillet Meta). Les boutons admin apparaissent dans la grille existante, au même niveau que "Mon portefeuille", "Mes réservations", etc.

A — Boutons intégrés dans la grille

Les boutons admin s'ajoutent directement dans la grille existante (row-cols-2), au même format que les autres boutons du compte.
