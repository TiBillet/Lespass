{% translate "Un commun numérique coopératif" %}

{% blocktranslate %} TiBillet est né sur le terrain, dans les festivals et les cafés associatifs de La Réunion. Pas dans un bureau d'études, mais dans les besoins concrets des lieux qui accueillent du public : vendre des billets sans frais cachés, gérer les adhésions, encaisser au bar, et fédérer les agendas entre structures voisines. {% endblocktranslate %}

{% blocktranslate %} Aujourd'hui, c'est une boîte à outils complète — billetterie, cashless, caisse enregistreuse, monnaie locale, budget contributif — conçue pour et par des associations culturelles. Zéro dark pattern, remboursement garanti, open source. {% endblocktranslate %}

{% blocktranslate %} TiBillet est porté par une SCIC — une coopérative d'intérêt collectif. Les décisions sont prises ensemble, lors de réunions ouvertes. Le code est libre (licence AGPLv3), mais ce qui fait un commun numérique, ce n'est pas juste la licence : c'est la communauté et la gouvernance qui l'entourent. {% endblocktranslate %}

{% blocktranslate %} Le principe fédératif est simple : chaque lieu garde son autonomie (son agenda, sa caisse, ses adhésions) et choisit avec qui se connecter. Une seule carte pour plusieurs lieux. Un réseau qui grandit par le bouche-à-oreille, pas par le marketing. {% endblocktranslate %}
