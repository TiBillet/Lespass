{% translate "Un commun numérique coopératif" %}

{% blocktranslate %} TiBillet est né sur le terrain, dans les festivals et les cafés associatifs de La Réunion. Pas dans un bureau d'études, mais dans les besoins concrets des lieux qui accueillent du public : vendre des billets sans frais cachés, gérer les adhésions, encaisser au bar, et fédérer les agendas entre structures voisines. {% endblocktranslate %}

{% blocktranslate %} Aujourd'hui c'est une boîte à outils complète — billetterie, cashless NFC multi-lieux, caisse enregistreuse, monnaie locale, monnaie temps, budget contributif — pensée pour et par les associations culturelles. Zéro dark pattern, zéro commission cachée, code source libre. {% endblocktranslate %}

{% blocktranslate %} TiBillet est porté par la Coopérative Code Commun (SCIC à trois collèges égaux : animateur·ices, contributeur·ices, utilisateur·ices). Le code est libre — mais ce qui fait un commun numérique, c'est la communauté qui l'entoure et la gouvernance partagée qui le pilote. {% endblocktranslate %}

{% blocktranslate %} Inspirés des travaux d'Elinor Ostrom : un dépôt Git avec une licence libre ne suffit pas. Il faut une ressource partagée, une communauté vivante, une gouvernance organisée. Le principe fédératif est simple — chaque lieu garde son autonomie (son agenda, sa caisse, ses adhésions) et choisit avec qui se connecter. Une seule carte pour plusieurs lieux. Un réseau qui grandit par la rencontre, pas par le marketing. {% endblocktranslate %}
